# ⚖️ Celedonio Advocacia & Consultoria | Landing Page Premium

Uma plataforma digital institucional de alta conversão desenvolvida exclusivamente para o mercado jurídico. O projeto une um design sóbrio e autoritário com gatilhos de acolhimento e segurança, respeitando rigorosamente o Código de Ética da OAB.

A Celedonio Advocacia não é apenas um site estático, é um **motor de triagem de clientes premium** para as áreas do Direito Civil, de Família e Penal.

---

## 🚀 Engenharia de Conversão & Features Jurídicas

Além do design premium (Azul Marinho e Dourado com tipografia *Via Appia*), a plataforma foi equipada com funcionalidades específicas para capturar leads no mercado legal:

- **🚨 Alerta de Urgência (Banner 24h):** Banner fixo no topo com gatilho direto para o WhatsApp, essencial para conversão imediata de causas criminais (flagrantes).
- **⏳ Timeline de Processos:** Uma seção visual animada que detalha o passo a passo da atuação do escritório, reduzindo a ansiedade do cliente e ancorando transparência.
- **📖 Glossário Interativo (Legal Tooltips):** Sistema de *tooltips* customizados que explicam jargões jurídicos (ex: *Liminar*, *Habeas Corpus*) ao passar o mouse, educando o cliente sem poluir a interface.
- **🤖 Triagem Inteligente via Chatbot:** Botão flutuante estilizado que não apenas leva ao WhatsApp, mas realiza um mini-funil de perguntas (Área de Atuação e Nível de Urgência) antes do redirecionamento.
- **🛡️ Conformidade LGPD & OAB:** Formulários com *checkbox* de consentimento explícito, protegendo o escritório e os dados do cliente (Provimento 205/2021).

## 🛠️ Stack Tecnológica

- **Framework:** Next.js 14 (App Router)
- **Linguagem:** TypeScript
- **Estilização:** Tailwind CSS
- **Animações:** Framer Motion
- **Ícones & UX:** Lucide React

## 🚀 Como Rodar o Projeto Localmente

**1. Clone o repositório:**
`git clone https://github.com/davidenisDEV/CeledonioAdvg.git`

**2. Acesse a pasta do projeto:**
`cd CeledonioAdvg`

**3. Instale as dependências:**
`npm install`

**4. Inicie o servidor local:**
`npm run dev`
